About Jonathan

I am a Mid-Senior Unity Developer from Ontario, Canada, with 8+ years of experience developing applications both personally and professionally.

    • Proficient in both 2D and 3D application development (worked on over 30 projects).
    • Specialize in mobile app optimization and dynamic UI system implementation.
    • Experienced in backend integration, including Firebase and API calls.
    • Always prioritize the three C's: Camera, Character, and Controls.
    • Skilled at maximizing the efficiency of Unity components, including:
      • Scriptable objects for data management.
      • Serializable classes and attributes for structured data storage.
      • Inheritance for scripts and prefab variants to enhance reusability.
      • Actions, asynchronous calls, and callback methods for efficient event handling.
      • Animation systems, 2D/3D lighting and particle systems to make games look and feel great.

Ultimate Ball Sort

This is a small game I challenged myself to make in a couple of days. The game was based on a real-life challenge from the TV reality series "Survivor". The game went viral on itch.io shortly after being launched.

    • Sorting logic
    • Drag 'n drop gameplay
    • Multiple game modes
    • High score saving
    • 3D collision detection
    • Random spawn systems
    • Custom designed 3D board model
    • PC and mobile optimization

Bubble Shooter

This game was a surprising technical challenge, not only due to its recursive algorithms for ball popping, but due to the requirement for the game to be embedded into a native Swift app, utilizing Unity as a Library (UAAL).

    • Recursive sorting algorithms
    • Random daily seeds
    • UAAL Integration into Swift
    • Pixel-perfect aiming-line
    • Popping/falling logic
    • Custom board-builder side app
    • Multi-ball colours and ball swapping
    • Two-way communication between Unity/Swift

2D Platformer

I teach aspiring game developers on YouTube how to make this simple 2D platformer game through a series of videos. I have prototyped all of the code and teach it in a proper, easy to comprehend manner.

    • isGrounded checks
    • Double jumps
    • Wall jumps
    • Tilemaps and random decorations
    • 2.5D parallax system
    • Cinemachine state-driven camera
    • Enemy AI navigation
    • Raycasting for collision detection

2D/3D Arkanoid Hybrid

This is a prototype I am developing, putting a new twist on the classic Arkanoid game. It Requires the player to swap between 2D and 3D gameplay to proceed.

    • 2D and 3D controls
    • Object pickup system
    • 2D and 3D mixed-physics
    • Probuilder level design
    • Baked and mixed lighting
    • Callback methods
    • 1st and 3rd person mixed gameplay
    • Reflection and light probes

    • In-app purchases and ads
